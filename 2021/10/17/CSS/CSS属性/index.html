CSS属性

1. 布局常用属性

  • width 设置元素标签的宽度，如：width:100px;
  • height 设置元素标签的高度，如：height:200px;
  • background 设置元素背景色，如：background:gold;
  • background:url(xxx) 设置元素的背景图片，如background:url(images/logo.png) no-repeat;不重复显示
  • border 设置元素四周的边框，如：border:1px solid black;
  • border-top 设置顶边边框
  • border-left 设置左边边框
  • border-right 设置右边边框
  • border-bottom 设置底部边框
  • padding 设置元素包含的内容和元素边框的距离，也叫内边距，如padding:20px;
  • padding-top 设置顶部内边距
  • padding-bottom 设置底部内边距
  • padding-left 设置左部内边距
  • padding-right 设置右部内边距
  • marin 设置元素和外界的距离，也叫外边距，如margin:20px;
  • margin-top 设置上部外边距
  • margin-bottom 设置底部外边距
  • margin-left 设置左部外边距
  • margin-right 设置右部外边距
  • float 设置元素浮动，浮动可以让块元素排列在一行。
  • float:left 左浮动
  • float:right 右浮动

2. 文本属性样式

  • color 设置文字的颜色，如：color:red;

  • font-size 设置文字的大小，如：font-size:12px;

  • font-family 设置文字的字体，如：font-family:‘Microsoft Yahei’;

  • font-weight 设置文字是否加粗，如：font-weight:bold;设置加粗，font-weight:normal;设置不加粗。

  • line-height 设置文字行高，如：line-height:30px，表示文字高度加上是文字上下的间距是30px。

  • text-decoration 设置文字的下划线，如text-decoration:none;去掉文字下划线。

  • text-align 设置文字水平对齐方式，如text-align:center; 设置文字水平居中。

  • text-indent 设置文字首行缩进，如text-indent:24px; 设置文字首行缩进24px。
