JavaScript数组

2. 数组的定义

3. 数组的操作

（1）获取数组的长度 array.length

（2）根据下表索引 array[index]

（3）从末尾删除或添加数据 array.push(element); array.pop(element);

（4）根据下标添加元素和删除元素

array.splice(start,num,element1,….,elementN)

参数解析：

  • start 必须，开始删除的索引
  • num 可选，删除数组元素的个数
  • elementN 可选，在start索引位置插入的新元素

此方法会删除从start索引开始的num个元素，并且将N个element插入到start位置。

JS变量和数据类型

发表于 2021-10-20

JS变量和数据类型

1. 定义变量

JavaScript是一种弱类型语言，也就是不需要指定变量的类型，变量的类型由它的值来决定。

定义变量需要使用’var’，一条JavaScript语句应该以分号;结尾。

定义变量语法格式：

var 变量 = 值;

2. JavaScript注释

JS的注释方法和C语言一样：

  • 单行注释 //
  • 多行注释 /* */

3. 数据类型

JavaScript中有六种数据类型，包括五种基本数据类型和一种复杂数据类型（object）。

五种基本数据类型：

  • number 数字类型
  • string 字符串类型
  • boolean 布尔类型
  • undefined 变量声明未初始化
  • null 空

符合数据类型（object）

4. 变量命名规范

（1）区分大小写；

（2）第一个字符必须是字母、下划线或者美元符号；

（3）其它字符可以是字母、下划线、美元符或数字。

5. 匈牙利命名法

对象o Object 比如：oDiv

数组a Array 比如：aItems

字符串s string 比如：sUserName

整数i integer 比如：iItemCount

布尔值b Boolean 比如：bIsComplete

浮点数f float 比如：fPrice

函数fn Function 比如fnHandler

JavaScript的介绍

1. JavaScript的定义

JavaScript是运行在浏览器的脚本语言，是由浏览器解释执行的，简称js，它能够让网页和用户有交互功能，增加良好的用户的体验效果。

前段开发三套件：

  • HTML 编写网页，负责网页结构
  • CSS 美化网页，负责网页样式
  • JavaScript 使网页具有交互功能，负责网页行为

2. JavaScript的使用方式

（1）行内式（主要用于事件）

直接写在标签的属性中；

（2）内嵌式

写在html文件的<head>部分

（3）外链式

单独开一个后缀名为.js的文件，存放js代码，h5中使用script src = “ “ 引用这个文件

盒子模型

1. 盒子模型介绍

所谓的盒子模型，就是把HTML页面的元素看做一个矩形的盒子，矩形盒子是由内容（content）、内边距（padding）、边框（border）、外边距（margin）四个部分组成。

2. 相关样式属性

  • 盒子的内容宽度（width），注意不是盒子的宽度
  • 盒子的内容高度（height），注意不是盒子的高度
  • 盒子的边框（border）
  • 盒子内的内容和边框之间的间距（padding）
  • 盒子外边距（margin）

3. 示例代码

设定边框：

border: 10px solid blueviolet;

设定内边距：

padding: 10px;

设定外边距：

margin: 10px;
